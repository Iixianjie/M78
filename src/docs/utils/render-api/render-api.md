# RenderApi 渲染api

有时候我们需要在react主实例外渲染组件，例如Modal、Drawer等组件，RenderApi为其提供了便捷的操作，组件只需要遵循几个api即可配合使用。

RenderApi在独立的包中维护，如果需要独立使用，请安装`@lxjx/react-render-api`

具体的使用见[react-render-api](https://github.com/Iixianjie/react-render-api)
